## Managing virtual environments

Creating a virtual environment.

```bash
python3 -m venv venv
```

Entering the virtual environment.

```bash
source venv/bin/activate
# or
venv-enter
```

Exiting the virtual environment.

```bash
deactivate
# or
venv-exit
```

Instaling Django

```bash
python -m pip install Django
```
